#### 使用说明

1.  下载本项目，直接下载或git clone 项目地址
2.  在数据库中运行sql/hrms.sql
3.  在idea中导入项目，点击运行即可
注意：maven中的依赖可能会由于网络原因下载失败，此时可在maven库中找到下载失败的依赖，删除后缀名为lastupdated的文件，再在idea中重新下载
